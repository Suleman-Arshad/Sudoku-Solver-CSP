# Sudoku Solver (AI & Interactive)

A modern, high-performance Sudoku Solver application built with **Python** and **Tkinter**. This project demonstrates how to solve complex puzzles using Artificial Intelligence techniques like Constraint Satisfaction Problems (CSP).

## 🚀 Features
- **AI Mode:** Watch the computer solve puzzles instantly using **AC-3** and **Backtracking** algorithms.
- **User Mode:** Play Sudoku yourself with real-time error detection.
- **Smart Hint System:** Stuck on a move? The AI can reveal the correct number for any cell or correct your mistakes.
- **Sleek UI:** A dark-themed, user-friendly interface with a vertical scrollbar for easy navigation.
- **Multiple Difficulties:** Load puzzles from `Easy`, `Medium`, or `Hard` categories.

## 🧠 Algorithms Used
- **AC-3 (Arc Consistency):** Reduces the possible values for each cell before searching.
- **Backtracking Search:** A powerful depth-first search algorithm that finds the solution by trial and error with constraint checking.

## 🛠️ Installation & Setup
1. **Prerequisites:** Make sure you have Python 3.x installed on your system.
2. **Clone the Project:**
   ```bash
   git clone [https://github.com/Suleman-Arshad/Sudoku-Solver-CSP.git](https://github.com/Suleman-Arshad/Sudoku-Solver-CSP.git)
   cd Sudoku-Solver-CSP
3. **Puzzle Files:** Ensure easy.txt, medium.txt, and hard.txt are in the same folder as the script.
## 🎮 How to Run
Run the application using the following command:
    ```bash
    python sudoku_solver.py
